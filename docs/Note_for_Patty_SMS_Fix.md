# Note for Patty – SMS / Text Message Fix

Hi Patty,

Here’s a short summary of what we fixed with the text messages from clients and donors.

**What was wrong**  
When someone texted us, the system was sometimes attaching their message to an **old Service** (e.g. from 2024) instead of their **current one** (e.g. from 2026). So when you opened their current Service, the new texts didn’t show there—they were still on the old Service.

**What we changed**  
We updated how the system decides which **Service** to link a text to. It now always uses the **most recent Service** for that person (by when it was created), instead of sometimes picking an older one.

**What you’ll see now**  
For **new** text messages, they should show up on the correct, current **Service**. So when you open “Katie 01/23/2026” (or whatever the current Service is), her new texts will be there.

**One caveat**  
Messages that were already linked to an old Service before this fix are still on those old records. We didn’t move those. Only **new** incoming texts use the new behavior.

If you see a new text still landing on an old Service, let us know and we can look at it.
